Offensive security curriculum

Learn to hack by hacking something built to be hacked.

Seven tracks, twenty-two hands-on levels. Break real vulnerability classes — injection, XSS, broken auth, CSRF/SSRF, crypto mistakes — in sandboxes built for exactly that, then carry the same instincts into methodology, tooling, and real bug bounty programs.

Start with Recon → Read the rules of engagement
Scope, for the record: every target here is a sandboxed simulation running entirely in your browser — nothing you do on this page touches a real network, real infrastructure, or another person. That containment is also why it's honest with you about its limits: it teaches the mechanics of each bug class, but real engagements need real (authorized) targets. Track 7 tells you exactly where to go next. Never point anything you learn here at a system you don't have explicit permission to test.
